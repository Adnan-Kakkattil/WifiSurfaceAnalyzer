Abstract: Wi-Fi Attack Surface Analyser
The widespread adoption of wireless networks has considerably increased the attack surface available to threat actors, making Wi-Fi infrastructures a critical target during reconnaissance and initial access phases of cyber attacks. Conventional wireless security tools tend to focus on individual vulnerabilities such as weak encryption or misconfigurations, while failing to provide a unified, attacker-centric view of wireless exposure. This project proposes an Advanced Wi-Fi Attack Surface Analyser that systematically identifies, analyzes, and prioritizes Wi-Fi security risks from a realistic adversarial perspective.
The system conducts passive wireless reconnaissance to collect information about access points, including SSID visibility, encryption protocols, channel usage, signal strength, and broadcast behavior. A context-aware risk assessment engine evaluates the collected data by correlating technical weaknesses with environmental factors such as signal leakage and network discoverability, enabling accurate prioritization of risks based on attack feasibility rather than static severity metrics.
The analyser is implemented using a Python-based technology stack, utilizing Python 3 as the core programming language, Scapy for low-level IEEE 802.11 packet analysis, and Linux-based wireless drivers operating in monitor mode for passive traffic capture. Security reporting and visualization are handled using the Rich library, while structured outputs are generated in JSON format to support automation and future integration. The modular design allows seamless extension with Cyber Threat Intelligence (CTI) feeds, machine learning models, and security operations workflows.
By combining passive reconnaissance, risk-based analysis, and an extensible Python-centric architecture, this project provides a research-oriented framework for understanding and reducing Wi-Fi attack surfaces in enterprise and public wireless environments.
